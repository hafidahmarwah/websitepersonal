HTML Dasar

Apa itu HTML?

HTML (Hypertext Markup Language) adalah bahasa standar yang digunakan untuk membuat halaman web. HTML digunakan untuk mendeskripsikan struktur sebuah halaman web menggunakan elemen-elemen atau tag HTML.

Struktur Dasar HTML

Struktur dasar sebuah dokumen HTML terdiri dari beberapa bagian penting, yaitu: DOCTYPE, html, head, dan body.

Berikut adalah contoh struktur dasar HTML:

<!DOCTYPE html>
<html>
  <head>
    <title>Halaman Pertama Saya</title>
  </head>
  <body>
    <h1>Selamat datang di Halaman Pertama Saya</h1>
    <p>Ini adalah paragraf pertama saya</p>
  </body>
</html>

Pada contoh di atas, kita dapat melihat beberapa elemen HTML yang penting:

  • <!DOCTYPE html>: Menentukan jenis dokumen HTML yang digunakan.
  • <html>: Menandakan awal dari dokumen HTML.
  • <head>: Bagian untuk menyertakan informasi tentang dokumen, seperti judul.
  • <title>: Menampilkan judul halaman yang muncul di tab browser.
  • <body>: Tempat di mana konten utama halaman, seperti teks, gambar, dan lainnya, diletakkan.
  • <h1>: Elemen untuk membuat heading atau judul utama.
  • <p>: Elemen untuk membuat paragraf teks.

Penjelasan tentang Elemen HTML

Elemen HTML adalah bagian dari struktur HTML yang mendefinisikan konten di dalam halaman web. Setiap elemen HTML biasanya terdiri dari pasangan tag pembuka dan penutup. Contoh elemen HTML yang sering digunakan antara lain:

  • <a>: Digunakan untuk membuat tautan atau link ke halaman lain.
  • <img>: Digunakan untuk menyisipkan gambar ke dalam halaman web.
  • <div>: Digunakan untuk membuat pembagian atau kontainer dalam halaman web.
  • <span>: Digunakan untuk memberi penekanan pada bagian tertentu dalam konten teks.
