PHP Dasar

Apa itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang digunakan untuk mengembangkan aplikasi web dinamis. PHP sering digunakan untuk membuat halaman web interaktif dan dapat berfungsi dengan berbagai database seperti MySQL.

Struktur Dasar PHP

PHP biasanya disisipkan ke dalam HTML dengan menggunakan tag khusus. Berikut adalah contoh struktur dasar PHP dalam halaman HTML:

<html>
  <head>
    <title>Halaman PHP Pertama</title>
  </head>
  <body>
    <?php
      echo "Selamat datang di halaman PHP pertama saya!";
    ?>
  </body>
</html>

Pada contoh di atas, kode PHP dimulai dengan <?php dan diakhiri dengan ?>, dan fungsi echo digunakan untuk menampilkan teks di halaman web.

Penjelasan tentang Elemen PHP

PHP digunakan untuk membuat halaman web dinamis dengan cara memproses data di server dan menghasilkan konten HTML yang ditampilkan di browser. Berikut adalah beberapa elemen penting dalam PHP:

  • <?php ... ?>: Tag pembuka dan penutup PHP di dalam dokumen HTML.
  • echo: Digunakan untuk menampilkan data atau teks ke halaman web.
  • variables: Variabel dalam PHP dimulai dengan tanda dolar ($) diikuti dengan nama variabel. Contoh: $nama = "John";
  • functions: PHP mendukung banyak fungsi built-in. Misalnya, fungsi strlen() digunakan untuk menghitung panjang string.

Contoh Penggunaan PHP

Berikut adalah contoh penggunaan PHP untuk memproses data formulir dan menampilkan pesan berdasarkan input pengguna:

<form method="post" action="php_example.php">
  Nama: <input type="text" name="nama">
  <input type="submit">
</form>

<?php
  if ($_SERVER["REQUEST_METHOD"] == "POST") {
      $nama = $_POST['nama'];
      echo "Halo, " . $nama;
  }
?>

Contoh di atas menunjukkan cara menggunakan formulir HTML untuk mengirim data ke server menggunakan metode POST, kemudian memproses dan menampilkan input pengguna menggunakan PHP.

PHP dengan MySQL

PHP sering digunakan bersama dengan MySQL untuk membuat aplikasi web yang melibatkan penyimpanan dan pengambilan data dari database. Berikut adalah contoh sederhana untuk menghubungkan PHP dengan database MySQL:

<?php
  $conn = new mysqli("localhost", "username", "password", "database");
  if ($conn->connect_error) {
      die("Koneksi gagal: " . $conn->connect_error);
  }
  echo "Koneksi berhasil!";
?>

Di sini, PHP digunakan untuk membuat koneksi ke database MySQL dan memverifikasi apakah koneksi berhasil.
