JavaScript Dasar

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif. JavaScript memungkinkan pengembang untuk menambah fungsionalitas pada halaman web, seperti menangani event pengguna, memanipulasi elemen DOM (Document Object Model), serta berinteraksi dengan server melalui AJAX.

Struktur Dasar JavaScript

JavaScript sering ditulis di dalam tag <script> dalam halaman HTML. Berikut adalah contoh dasar penggunaan JavaScript:

<html>
  <head>
    <title>Halaman JavaScript Pertama</title>
  </head>
  <body>
    <h1>Selamat datang di JavaScript</h1>

    <script>
      alert("Halo, ini adalah pesan dari JavaScript!");
    </script>
  </body>
</html>

Pada contoh di atas, alert() adalah fungsi JavaScript yang digunakan untuk menampilkan pesan pop-up pada layar.

Penjelasan tentang Elemen JavaScript

Beberapa elemen dasar dalam JavaScript yang perlu Anda ketahui adalah:

  • Variables: Variabel digunakan untuk menyimpan data. Dalam JavaScript, kita dapat menggunakan let, const, atau var untuk mendeklarasikan variabel.
    • let nama = "John";
const umur = 30;
  • Functions: Fungsi adalah blok kode yang dapat dipanggil untuk melakukan tugas tertentu. Berikut adalah contoh mendeklarasikan fungsi dalam JavaScript:
    • function halo() {
  alert("Selamat datang di JavaScript!");
}
halo(); // Memanggil fungsi halo
  • Events: JavaScript memungkinkan kita untuk menangani berbagai event, seperti klik tombol atau perubahan input pengguna. Misalnya, kita bisa menggunakan event onclick untuk menangani klik tombol.
    •
  • DOM Manipulation: JavaScript memungkinkan kita untuk memanipulasi elemen-elemen HTML menggunakan DOM. Berikut adalah contoh untuk mengubah teks dalam elemen dengan ID tertentu:
    • document.getElementById("demo").innerHTML = "Teks telah diubah!";

Contoh Penggunaan JavaScript

Berikut adalah contoh penggunaan JavaScript untuk menampilkan teks ketika tombol diklik:

<button onclick="tampilkanPesan()">Klik untuk Pesan</button>
<p id="pesan"></p>

<script>
  function tampilkanPesan() {
    document.getElementById("pesan").innerHTML = "Tombol telah diklik!";
  }
</script>

Pada contoh di atas, ketika pengguna mengklik tombol, JavaScript akan mengubah isi elemen dengan ID pesan menjadi "Tombol telah diklik!".

JavaScript dan AJAX

JavaScript juga dapat digunakan untuk melakukan komunikasi asinkron dengan server tanpa memuat ulang halaman web menggunakan AJAX (Asynchronous JavaScript and XML). Berikut adalah contoh sederhana penggunaan AJAX:

<script>
  function ambilData() {
    var xhttp = new XMLHttpRequest();
    xhttp.onreadystatechange = function() {
      if (this.readyState == 4 && this.status == 200) {
        document.getElementById("data").innerHTML = this.responseText;
      }
    };
    xhttp.open("GET", "data.txt", true);
    xhttp.send();
  }
</script>

<button onclick="ambilData()">Ambil Data</button>
<p id="data"></p>

Dalam contoh ini, saat tombol diklik, JavaScript menggunakan AJAX untuk mengambil data dari file "data.txt" dan menampilkannya pada elemen dengan ID data.
