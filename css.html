CSS Dasar

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain atau memberikan gaya pada halaman web. CSS memungkinkan Anda untuk mengatur warna, layout, font, jarak, dan banyak elemen desain lainnya.

Struktur Dasar CSS

CSS digunakan dengan menuliskan aturan (rule) yang terdiri dari selector, properti, dan nilai. Berikut adalah contoh aturan CSS:

selector {
    property: value;
}

Contoh aturan CSS yang mengubah warna teks menjadi biru:

h1 {
    color: blue;
}

Pada contoh di atas, h1 adalah selector yang merujuk pada elemen <h1>, dan color: blue adalah properti yang memberi nilai warna biru pada teks dalam elemen tersebut.

Penjelasan tentang Elemen CSS

CSS terdiri dari beberapa elemen penting yang digunakan untuk mendesain halaman web, di antaranya:

  • Selector: Menentukan elemen HTML mana yang akan diberi gaya.
  • Property: Menentukan gaya atau fitur yang ingin diterapkan pada elemen yang dipilih.
  • Value: Nilai yang diterapkan pada properti untuk menentukan tampilannya.

Contoh Penggunaan CSS

Berikut adalah contoh penggunaan CSS di dalam dokumen HTML. CSS dapat disertakan di dalam dokumen HTML dengan tiga cara:

  • Inline CSS: Menyisipkan CSS langsung di dalam elemen HTML menggunakan atribut style.
    • <p style="color: red;">Teks ini berwarna merah</p>
  • Internal CSS: Menambahkan kode CSS di dalam tag <style> di bagian <head> dokumen HTML.
    • <head>
    <style>
        p {
            color: green;
        }
    </style>
</head>
  • External CSS: Menyimpan kode CSS dalam file terpisah yang kemudian dipanggil melalui tag <link>.
    • <link rel="stylesheet" href="style.css">
